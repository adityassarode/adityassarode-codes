## **Experiment 1:**

### **Programs Included:**
- a) Program to calculate the average of three numbers entered by the user.

- b) Program to calculate the factorial of a given number.

- c) Program to generate and print the first n numbers of the Fibonacci series.

- d) Program to convert temperature from Fahrenheit to Celsius and vice versa.

- e) Program to print a simple pattern (triangle or square).

- f) Program to implement a number guessing game where the user guesses a randomly generated number.

---

## **Experiment 2:**

### **Programs Included:**
- a) Basic calculator performing addition, subtraction, multiplication, and division.

- b) Program to check whether a string is a palindrome.

- c) Program to count the frequency of each character in a given string.

- d) Java application for string manipulation (concatenation, substring extraction, and searching).

- e) Java program to read a text file and display the frequency of each word.

- f) Java program to handle file I/O, reading data from one file and writing it to another.

---

## **Experiment 3:**


### **Programs Included:**
- a) Implementation of a Sortable interface with methods to sort arrays of different data types.

- b) Java class representing a Car, with attributes such as model and year, and methods to display information.

- d) Java application to generate a random password based on user-defined criteria (length, character types).
